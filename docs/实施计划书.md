## 一、项目概述

本项目实现一套 **3D 点云目标分割系统**：读取 3D 点云文件并三维绘制，估计每个点的法向量与
曲率，采用**区域生长**算法将点云按表面朝向的一致性分割成若干光滑片段，并通过逐层收紧阈值
实现**由粗到细**的迭代细分；最终在交互式三维界面中用不同颜色显示不同片段，并支持回放由粗
到细的各个阶段。

选题难点在于：(1) 需要图形学与 OpenGL 基础；(2) 分割算法需综合考虑表面朝向、点的位置分布、
局部连通性与片段一致性/差异性度量；(3) 由粗到细的迭代过程及其可视化。

| F6 | 三维可视化 | OpenGL 绘制；旋转/缩放/平移、三视图、点大小调节 |
| F7 | 分色与回放 | 片段分色；滑块在各层间切换回放 |

## 三、技术路线

- **语言/构建**：C++17 + CMake；面向对象组织（数据/算法/显示分层解耦）。
- **算法自实现**：KD 树、PCA（含 3×3 对称矩阵 Jacobi 特征分解）、区域生长、由粗到细
  分层细分。**不依赖 PCL**，以体现对算法本身的理解。
- **图形界面**：GLFW（窗口/输入）+ OpenGL 3.3（渲染）+ Dear ImGui（控制面板）。
  （原计划 Qt，因开发机 macOS 12 无法安装 Qt6 而改用更轻量、可在本机编译的方案。）
- **文档**：实习报告用学校 `nwafupaper` LaTeX 模板，XeLaTeX 编译。

## 四、系统设计概要

处理流程：`点云文件 → KD树 → PCA法向量+曲率 → 区域生长(粗) → 逐层细分 → 分段着色 → OpenGL视口`。

主要类：`Vec3`、`PointCloud`、`KdTree`、`SymEigen`(特征分解)、`Segmentation`(区域生长/由粗到细)、
`PointCloudIO`、`Camera`(轨迹球相机)、`Viewer`(渲染+界面)。核心算法库 `pcseg_core` 不依赖
图形界面，可被命令行、测试、界面三者共享。

## 七、测试计划

- **单元测试**（CTest）：KD树近邻(对比暴力解)、平面法向量正确性、PLY 读写往返、
  区域生长在合成场景上的段数与纯度。
- **集成验证**：命令行在演示场景跑通全流程并导出着色点云；界面对各参数实时分割。
- **性能**：统计上万点规模下法向量估计 + 分割的耗时与交互帧率。
